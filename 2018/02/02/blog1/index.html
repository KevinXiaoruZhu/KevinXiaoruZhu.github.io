小米路由器1/2/3科学上网配置方法(新)

2018年02月02日 |
计算机
| 路由器 科学上网

插件名称: Misstar Tools
路由器支持: 小米路由器硬盘版1/2（R1D&R2D），小米路由器3（R3）
类型: ROOT SSH
开发: Starry

必备工具

  • 小米路由器开发版(建议手动安装较低版本2.11.20,下载地址)
  • SSH功能(具体步骤请戳这里开启ssh)

步骤

  1. 开启SSH后在SSH界面使用wget方法安装工具箱一键包:

    $ wget http://www.misstar.com/tools/appstore/install.sh -O /tmp/install.sh && chmod +x /tmp/install.sh && /tmp/install.sh

  2. 安装成功后进入路由器后台管理界面(miwifi.com), 进入“MT工具箱”中的“插件管理”:

  3. 由于某些不可描述的原因，插件列表里并没有「科学上网(ss)」，但是仍可在插件中心里找任意一个插件(以webshell为例):

  4. 以查看网页源代码的方法找到「安装」按钮代码的id(右键-Inspect):

  5. 把此id改为“ss”:

  6. 点击「安装」按钮进行安装,成功后打开「科学上网」，添加节点输入配置信息，选择游戏模式(gamemode)即可实现自动代理上网:
